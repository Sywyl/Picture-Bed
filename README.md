# Open source Picture bed

使用开源上传组件[ bootstrap-fileinput ](https://github.com/kartik-v/bootstrap-fileinput/)用 Spring Boot 写了一个图床

> 目前支持：七牛云，阿里云


### Configuration

> #### 使用的时候需要配置`application.properties` 即可
